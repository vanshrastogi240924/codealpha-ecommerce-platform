# ShopEasy - Online Shopping Platform

A modern, highly responsive, single-page e-commerce storefront prototype. This project features a clean user interface built with Tailwind CSS, showcasing responsive layouts, dynamic filtering structures, and an interactive shopping cart.

---

## 🚀 Features

*   **Responsive Navigation & Banner:** Features a sticky top navigation bar with a responsive search layout, user profile links, and an eye-catching promotional hero banner.
*   **Dynamic Shopping Cart:** Fully functional client-side cart system that increments a visual notification badge with custom bounce animations when items are added.
*   **Interactive Product Grid:** Displays trending products with hover lifting effects (`translateY`), custom price layouts, review rating stars, and individual "Add to Cart" triggers.
*   **Categories Carousel:** A horizontally scrollable category selection bar tailored for seamless mobile and desktop browsing.
*   **Clean Tailwind UI:** Built entirely using modern utility-first CSS styling, ensuring smooth hover states and transition animations.

---

## 🛠️ Tech Stack

*   **HTML5:** Structured semantic markup for a multi-section storefront layout.
*   **Tailwind CSS:** Fast, utility-first styling for layout configurations, spacing, and modern interactive elements.
*   **JavaScript (ES6):** Powering the cart logic, DOM selection events, and interactive alert confirmations.

---

## 📦 How to Run the Project

This storefront runs entirely in the browser without requiring a backend server or heavy installation processes.
